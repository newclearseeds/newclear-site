Australian ASD Essential Eight

The Australian Signals Directorate’s Essential Eight is a practical baseline for reducing cyber risk. We help you design, implement, and continuously improve these controls to meet your business needs and regulatory expectations.

Essential Eight is evolving. In June 2026, ASD began consultation on a proposed new Essentials series that would expand the current framework for contemporary technology environments. Essential Eight remains the current framework while consultation continues, and we will help clients align existing controls with the new guidance as it develops. Read the ASD announcement.

Application Control

Allow only approved applications to run, blocking untrusted or malicious software and reducing the attack surface.

Patch Applications

Keep key applications updated to close known vulnerabilities that attackers routinely exploit.

Configure Office Macro Settings

Restrict macro execution to prevent malicious documents from running code on your systems.

User Application Hardening

Harden browsers and common apps to limit risky features and reduce exposure to web‑based threats.

Restrict Administrative Privileges

Apply least‑privilege access and privileged access management to prevent abuse of admin rights.

Patch Operating Systems

Keep OS updates current to close critical security gaps across endpoints and servers.

Multi‑factor Authentication

Require MFA for users and administrators to reduce the impact of stolen credentials.

Regular Backups

Implement secure, tested backups to recover quickly from ransomware or data loss incidents.
